# 公司設立與法務清單

## 一、公司設立（台灣）

### 設立形式建議
- **股份有限公司**（推薦）：適合未來募資、股權分配
- 資本額建議：NT$ 1,000,000-5,000,000

### 設立步驟
1. [ ] 公司名稱預查（經濟部商業司）
2. [ ] 銀行開立籌備處帳戶 + 驗資
3. [ ] 向所在地登記機關申請設立登記
4. [ ] 辦理稅籍登記（國稅局）
5. [ ] 勞健保加保（勞保局/健保署）
6. [ ] 開立公司銀行帳戶

### 必備文件
- [ ] 公司章程
- [ ] 股東同意書 / 發起人同意書
- [ ] 董事/監察人名單
- [ ] 設立登記申請書

---

## 二、股權規劃

### 創辦人股權建議
```
建議預留股權池：
├── 創辦人：60-70%
├── 員工股票選擇權（ESOP）：10-15%
└── 投資人保留：20-25%
```

### 股權保護機制
- 反稀釋條款
- 優先清算權
- 股份到期（Vesting）：4年 + 1年懸崖（Cliff）

## 四、資安合規

### 個人資料保護法（台灣）
- 蒐集個資需有明確目的
- 建立個資管理程序
- 個資外洩通報機制

### ISO 27001（建議取得）
- 提高企業客戶信任
- 政府標案常見要求
